    Here is again thier list with their description:

    1. <div> - A container used for layout and grouping elements.
    2. <p> - Defines a paragraph of text.
    3. <a> - Creates a hyperlink to another page or resource.
    4. <img> - Embeds an image in the document.
    5. <span> - An inline container used for styling a part of text.
    6. <h1> to <h6> - Heading elements from most to least important.
    7. <ul> - Defines an unordered (bulleted) list.
    8. <ol> - Defines an ordered (numbered) list.
    9. <li> - Defines a list item in ordered or unordered lists.
    10. <input> - Defines an input field for user data in a form.




    Importance of memorizing html-elments:

    Memorizing HTML elements is essential for building efficient and well-structured web pages. 
When you know the tags by heart, you can write code faster and focus more on design and functionality rather than constantly looking up syntax. 
This improves productivity and helps you troubleshoot and debug more effectively.

Understanding and recalling elements like div, span, form, table, and input allows you to structure content properly and build interactive layouts. 
As you build more projects, these elements become second nature, giving you confidence to combine them with CSS and JavaScript. 
In short, memorizing HTML tags forms the solid foundation every web developer needs.
